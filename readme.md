Przykładowy czat 

Aby uruchomić serwer

```bash
cd server
python httpchat.py
```

Wysyłka wiadomości
```bash

curl -X POST -d '{"text": "Hello World"}' http://{ip_address}:8888/chat
```

Pobranie wiadomości
```bash

curl -X POST -d '{"last_message_id":-1}' http://{ip_address}:8888/messages | python -m json.tool
```
